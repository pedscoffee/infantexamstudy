# Contributing to the Infant Physical Exam Study Reference

Thank you for your interest in improving this resource! Community contributions are what will make this chart genuinely useful for students and residents.

## How to Contribute

### Suggesting Content Changes

The easiest way to contribute is through the [Discussions page](https://github.com/pedscoffee/infantexamstudy/discussions/1):

- Suggest new findings or sections
- Point out errors or outdated information
- Share board-relevant pearls or mnemonics
- Recommend image references

### Reporting Issues

Use [GitHub Issues](https://github.com/pedscoffee/infantexamstudy/issues) to report:

- Bugs in the study mode, tutor, or editing features
- Display or layout problems
- Accessibility issues

### Submitting Code Changes

1. Fork the repository
2. Make your changes to `index.html`
3. Test in a browser (no build step required — just open the file)
4. Submit a pull request with a clear description of your changes

## Content Guidelines

### Accuracy

- All clinical content should be verifiable against standard pediatric references
- Cite sources when adding new findings or modifying management recommendations
- When in doubt, err on the side of including the most commonly tested/accepted approach

### Style

- **Finding names**: Use the most commonly recognized clinical term
- **Management**: Start with the most important/urgent step; keep it concise
- **Board Pearls**: Focus on what differentiates the finding, classic associations, and high-yield mnemonics
- **Critical flag**: Mark rows as high priority if the finding requires urgent intervention or is a frequently tested board topic

### Tone

- Write for a medical student or junior resident audience
- Be concise but not cryptic — brief explanations are more useful than keyword lists
- It's OK to be conversational in board pearls (e.g., "the boards love this one")

## Code of Conduct

- Be respectful and constructive in all interactions
- Focus on improving the resource for learners
- Remember that this is an educational tool — clinical decisions should always involve proper medical judgment
